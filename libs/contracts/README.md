# Contracts Library

The Contracts library provides contract definitions and types used for inter-service communication within the local food delivery platform. This library ensures that all services adhere to the same data structures and protocols, facilitating seamless integration and communication.

## Installation

To use the Contracts library in your project, you can install it via npm or yarn:

```bash
npm install @local-food-delivery-platform/contracts
```

or

```bash
yarn add @local-food-delivery-platform/contracts
```

## Usage

Import the necessary contracts and types in your service or application:

```typescript
import { Order, User, Shop } from '@local-food-delivery-platform/contracts';
```
