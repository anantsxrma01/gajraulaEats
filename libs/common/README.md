# Common Library

The Common library contains shared utilities and types that are used across various applications and services within the local food delivery platform. This library aims to promote code reusability and maintainability by centralizing common functionalities.

## Installation

To install the Common library, run:

```
npm install <library-name>
```

## Usage

Import the necessary utilities or types from the Common library in your application or service:

```typescript
import { utilityFunction, CommonType } from '@local-food-delivery-platform/common';
```
